# nmap-xml-csv

```
python nmap_xml_to_csv.py -h
usage: nmap_xml_to_csv.py [-h] xml_file csv_file

Парсинг Nmap XML в CSV формат.

positional arguments:
  xml_file    Путь к XML файлу, сгенерированному Nmap.
  csv_file    Путь к выходному CSV файлу.

options:
  -h, --help  show this help message and exit
```
